Projeto Final - Meu Primeiro Ambiente VR (Unity)

Requisitos:
- Cena com chão (Plane) e skybox configurado
- Pelo menos 5 objetos 3D posicionados na cena
- Projeto organizado com nomenclatura clara e pastas lógicas
- Configuração para Android (Meta Quest) preparada no projeto (Build Target Android + XR configurável)
- A cena pode ser visualizada e testada no PC pelo Unity Editor (Play), sem depender do óculos VR

Como executar no PC:
1) Abra o projeto no Unity Hub
2) Abra a cena principal em Assets/cena (ou Assets/Scenes)
3) Clique em Play para visualizar a cena
